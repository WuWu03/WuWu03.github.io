生成文章唯一链接


  • Hexo 的默认文章链接格式是年，月，日，标题这种格式来生成的。如果你的标题是中文的话，那你的 URL 链接就会包含中文，复制后的 URL 路径就是把中文变成了一大堆字符串编码，如果你在其他地方用这边文章的 url 链接，偶然你又修改了改文章的标题，那这个 URL 链接就会失效。为了给每一篇文章来上一个属于自己的链接，利用 hexo-abbrlink 插件 ,来解决这个问题。
